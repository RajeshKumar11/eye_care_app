# App Icons

Place your app icons in this directory:

- `app_icon.png` - PNG icon for macOS/Linux tray (recommended: 32x32 or 64x64)
- `app_icon.ico` - ICO icon for Windows tray

For generating app icons, you can use tools like:
- https://www.appicon.co/
- flutter_launcher_icons package

Recommended icon specifications:
- Android: 192x192 (xxxhdpi)
- iOS: 1024x1024
- Windows: 256x256 ICO
- macOS: 1024x1024 PNG
